# Client Product Management App

## Overview
This is a Windows Forms application built using C#, XAML, and .NET Framework. The purpose of this application is to manage client product information through functionalities such as importing, searching, adding, deleting, and exporting records stored in a CSV file.

## Features
- Import client product information from a CSV file.
- Display records on the user interface.
- Search for records using sequential and binary search algorithms.
- Add and delete records with insertion and deletion algorithms.
- Export updated records and save to a CSV file.
